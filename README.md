## Получение данных из API и сохранение в локальную базу данных   
   
Щас будет инструкция типа серьезный проект   
   
## Как запустить проект:

1. Установите зависимости   

```bash
npm install
```

2. Запустите проект   

```bash
npx expo start
```

3. Киньте мне деньги на банковскую карту
